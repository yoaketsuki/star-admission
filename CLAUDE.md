# 繁星推薦錄取標準儀表板 — 專案 Memory

## 專案簡介

台灣大學繁星推薦入學的歷年錄取標準查詢儀表板，涵蓋 110～115 學年度資料。

- **技術棧**：純前端（HTML + CSS + JavaScript），無框架，使用 Chart.js 繪圖
- **主要頁面**：`index.html`（同 `繁星推薦儀表板.html`，兩者內容相同）

---

## 檔案結構

| 檔案 | 說明 |
|------|------|
| `index.html` | 主頁面（144 行，但已 minify，實際內容龐大） |
| `data.json` | 所有校系錄取資料，17,836 筆，JSON array |
| `dept_order.json` | 各校科系的顯示排序，key = 學校名稱，value = 科系名稱 array |
| `繁星錄取標準_110~115.csv` | 各學年度原始 CSV（逐年分開） |
| `繁星錄取標準_全部.csv` | 合併的完整 CSV |

---

## data.json 資料結構

每筆記錄代表一個校系某一學年度的錄取資料：

```json
{
  "y":  110,           // 學年度
  "s":  "國立臺灣大學",  // 學校名稱
  "c":  "00101",        // 校系代碼
  "n":  "中國文學系",    // 科系名稱
  "q":  6,              // 招生名額
  "a":  6,              // 錄取人數
  "r1": 2,              // 第一輪錄取人數
  "p1": 3,              // 第一輪最低在校排名 %
  "r2": 4,              // 第二輪錄取人數
  "p2": 6,              // 第二輪最低在校排名 %
  "ek1": "頂標",        // 國文檢定
  "ek2": "前標",        // 英文檢定
  "ek3": "均標",        // 數A檢定（若有）
  "ek4": "均標",        // 數B檢定（若有）
  "ek5": "前標",        // 社會或自然檢定
  "ek6": "A級",         // 英聽檢定（若有）
  "bx": [               // 比序項目 [[名稱, 第一輪標準, 第二輪標準], ...]
    ["在校學業", "3%", "6%"],
    ["學測英文", null, "15"]
  ]
}
```

### 學測檢定欄位對照

| key | 科目 |
|-----|------|
| ek1 | 國文 |
| ek2 | 英文 |
| ek3 | 數A |
| ek4 | 數B |
| ek5 | 社會 / 自然（依科系） |
| ek6 | 英聽 |

---

## dept_order.json 結構

```json
{
  "國立臺灣大學": ["醫學系", "中國文學系", "外國語文學系", ...],
  "國立臺灣師範大學": [...]
}
```

用於在儀表板中按官方順序排列科系。

---

## CSV 原始資料欄位

```
學年度,學校,校系代碼,校系名稱,是否外加,招生名額,錄取人數,
第一輪錄取人數,第一輪在校排名%,第二輪錄取人數,第二輪在校排名%,
國文檢定,英文檢定,數A檢定,數B檢定,社會檢定,自然檢定,英聽檢定,比序
```

## 開發注意事項

- `index.html` 是 minified 的單頁應用，修改時需注意不要破壞壓縮格式
- 資料更新時同步更新 `data.json` 和對應年度的 CSV
- `dept_order.json` 的科系順序需與教育部官方順序一致
- 新增學年度資料時，需同時在 `data.json` 新增並新增對應 CSV 檔案
